Prim Algorithm for Mimimum Spanning Tree

I present here a Java based demonstration of the Prim algorithm that builds a minimum spanning tree for an undirected connected graph equipped with an edge weight.
I follow closely the approach of Cormen in his classical textbook.

Explanations

An undirected graph is randomly created. Then the largest connected component is extracted using Depth First Search algorithm.
Each edge is labeled with its own weight, also randomly generated.
Then the Prim algorithm is applied with all intermediate results saved as a collection.
This collection is sent to the browser as a JSON object and used for animation.
At each step all edges that belong to the tree are drawn in blue, the other edges are drawn in black.
Note that an edge can be removed from the tree if a lower cost can be obtained by a different path. The partial cost is displayed at each step.
